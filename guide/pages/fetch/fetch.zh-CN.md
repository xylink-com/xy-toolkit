# Fetch 

类 Axios API 的网络请求工具。

# 特性

- [ ] 从浏览器中创建 XMLHttpRequests
- [ ] 从 node.js 创建 http 请求
- [ ] 支持 Promise API
- [ ] 拦截请求和响应
- [ ] 转换请求数据和响应数据
- [ ] 取消请求
- [ ] 自动转换 JSON 数据
- [ ] 客户端支持防御 XSRF


## Browser Support

- Chrome
- Firefox
- Safari 6.1+
- Internet Explorer 10+

Note: modern browsers such as Chrome, Firefox, Microsoft Edge, and Safari contain native implementations of window.fetch.
